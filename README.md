# LokiCheckout_DotdigitalgroupEmail

**This Magento 2 module is an add-on package to the LokiCheckout.**

## Installation
Install this package via composer:
```bash
composer require loki-checkout/magento2-dotdigitalgroup-email
```

Next, enable this module:
```bash
bin/magento module:enable LokiCheckout_DotdigitalgroupEmail
```
